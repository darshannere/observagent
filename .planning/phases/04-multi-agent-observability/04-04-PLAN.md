curl -s -X POST http://localhost:4999/ingest -H 'Content-Type: application/json' \

**Check 1 — Layout (AGENT-01):** The dashboard shows a 3-column layout. The leftmost column is an "Agent Tree" panel showing the agent hierarchy as an indented list. The tool call log is in the center.
