# Plan 10-03: AgentTree Enhancements

## Requirements
- AGNT-01: User can see the agent hierarchy as a collapsible tree (expandable/collapsible per branch)
- AGNT-03: Each agent row displays a human-readable name in the format `description [short-id]`
- AGNT-04: Each agent row shows the current tool being executed in real time

## Tasks

### Task 1: Add localStorage-backed collapse state
- **File**: `frontend/src/components/AgentTree.tsx`
- **Action**:
  1. Create localStorage key constant: `observagent:collapsed-sessions`
  2. Add useEffect to load initial collapse state from localStorage on mount
  3. Add toggle function that updates localStorage and state
  4. Pass collapse state to each session group via open attribute
- **Verification**: Collapsing a session, refreshing the page, and returning to dashboard shows session still collapsed

### Task 2: Implement human-readable names
- **File**: `frontend/src/components/AgentTree.tsx`
- **Action**:
  1. Extract agentType from the agent data
  2. Get last 4 characters of agentId: `agent.agentId.slice(-4)`
  3. Format as: `{agentType} [{last4}]` e.g., `gsd-executor [a1b2]`
  4. Display in the agent row instead of raw session ID
- **Verification**: Agent rows show readable names like "gsd-executor [a1b2]"

### Task 3: Display current tool in agent rows
- **File**: `frontend/src/components/AgentTree.tsx`
- **Action**:
  1. Read currentTool from the agent in the store
  2. Display tool name in the row (e.g., as a badge or inline text)
  3. Update on every PreToolUse event (already wired in Plan 10-02)
- **Verification**: Active agent rows show current tool name, updates on tool change

### Task 4: Wire up click to open detail panel
- **File**: `frontend/src/components/AgentTree.tsx`
- **Action**:
  1. Add onClick handler to agent row
  2. Call `setSelectedAgent(agentId)` on click
  3. Add visual feedback (cursor pointer, hover state)
- **Verification**: Clicking an agent row sets selectedAgent in store
