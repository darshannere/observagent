# Plan 10-05: LiveDashboard Integration

## Requirements
- AGNT-02: User can see a live active agent count badge showing how many agents are currently running
- AGNT-05: User can click an agent row to open a per-agent detail panel

## Tasks

### Task 1: Add active count badge to header
- **File**: `frontend/src/pages/LiveDashboard.tsx`
- **Action**:
  1. Add selector to compute active agent count: `Array.from(agents.values()).filter(a => a.state === 'active').length`
  2. Add badge in header area showing count with "active" label
  3. Style badge with green background when > 0, hidden when 0
  4. Position in header next to page title or filters
- **Verification**: Badge shows correct count, updates in real-time

### Task 2: Wire up AgentDetailPanel to LiveDashboard
- **File**: `frontend/src/pages/LiveDashboard.tsx`
- **Action**:
  1. Import AgentDetailPanel component
  2. Read selectedAgent from store
  3. Conditionally render AgentDetailPanel when selectedAgent is not null
  4. Pass agentId to panel and handle close callback
- **Verification**: Clicking agent row opens detail panel

### Task 3: Ensure real-time badge updates
- **File**: `frontend/src/pages/LiveDashboard.tsx`
- **Action**:
  1. Verify that active count updates when agent state changes via SSE
  2. Badge should reflect current active count without page refresh
  3. Test: Start an agent, badge should increment; complete agent, badge should decrement
- **Verification**: Badge updates in real-time with agent state changes

### Task 4: Layout integration
- **File**: `frontend/src/pages/LiveDashboard.tsx`
- **Action**:
  1. Ensure AgentDetailPanel overlays or resizes existing content

- **Verification**: Panel integrates smoothly without breaking existing layout
