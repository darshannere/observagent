<!-- API response shapes from Phase 12 (12-02-SUMMARY.md) -->

GET /api/insights/activity?session_id={id}
Response: Array<{ bucket_ms: number, tool_calls: number }>
  - bucket_ms: Unix epoch ms snapped to minute boundary (e.g. 1709000000000)
  - tool_calls: count of PostToolUse events in that minute
  - Returns [] for sessions with no data
  - Returns 400 { error: 'session_id required' } if session_id absent

GET /api/insights/tokens-over-time?session_id={id}
Response: Array<{ bucket_ms: number, input_tokens: number, output_tokens: number }>
  - bucket_ms: Unix epoch ms snapped to minute boundary
  - input_tokens: sum of input tokens in that minute
  - output_tokens: sum of output tokens in that minute
  - Returns [] for sessions with no data
  - Returns 400 { error: 'session_id required' } if session_id absent

<!-- Session ID derivation — from useObservStore -->
Store shape: sessions: Map<string, Session>
  Session: { sessionId: string, children: string[], cost: number, tokens: number }

To get the latest active session_id: derive from the sessions Map.
Use the first session_id in sessionCosts (most recently updated) as the proxy for "latest active session":

const [tokensData, setTokensData] = useState<{ bucket_ms: number; input_tokens: number; output_tokens: number }[]>([])
const [tokensStatus, setTokensStatus] = useState<'idle' | 'loading' | 'ok' | 'error'>('idle')

**Polling useEffect** — fires when activeTab changes to 'activity' or latestSessionId changes:
```typescript
useEffect(() => {
  if (activeTab !== 'activity' || !latestSessionId) return

  const fetchData = () => {
    setActivityStatus('loading')
    setTokensStatus('loading')
    fetch(`/api/insights/activity?session_id=${latestSessionId}`)
      .then(r => r.json())
      .then(d => { setActivityData(d); setActivityStatus('ok') })
      .catch(() => setActivityStatus('error'))

    fetch(`/api/insights/tokens-over-time?session_id=${latestSessionId}`)
      .then(r => r.json())
      .then(d => { setTokensData(d); setTokensStatus('ok') })
      .catch(() => setTokensStatus('error'))
  }

  fetchData()
  const interval = setInterval(fetchData, 30000)
  return () => clearInterval(interval)
}, [activeTab, latestSessionId])
```
