  <name>Task 1: Add activity endpoint (tool calls per minute)</name>
  <files>routes/insights.js</files>
  <action>
Append to the `insightsRoutes` function in `routes/insights.js`.

**Prepared statement** (add at the top of the function alongside existing statements):
```js
const stmtActivity = db.prepare(`
  SELECT
    (timestamp / 60000) * 60000 AS bucket_ms,
    COUNT(*) AS tool_calls
  FROM events
  WHERE session_id = ?
    AND hook_type = 'PostToolUse'
  GROUP BY bucket_ms
  ORDER BY bucket_ms ASC
`);
```

`(timestamp / 60000) * 60000` uses integer division (SQLite truncates) to snap each event timestamp to the start of its minute bucket, expressed in milliseconds.

**Route handler:**
```js
fastify.get('/api/insights/activity', (request, reply) => {
  const { session_id } = request.query;
  if (!session_id) return reply.code(400).send({ error: 'session_id required' });
  reply.send(stmtActivity.all(session_id));
});
```

Response shape: `[{ bucket_ms: 1741564800000, tool_calls: 7 }, ...]`

  <name>Task 2: Add tokens-over-time endpoint (input + output tokens per minute)</name>
  <files>routes/insights.js</files>
  <action>
Append to the `insightsRoutes` function in `routes/insights.js`.

**Prepared statement** (add at the top alongside existing statements):
```js
const stmtTokensOverTime = db.prepare(`
  SELECT
    (timestamp_ms / 60000) * 60000 AS bucket_ms,
    SUM(input_tokens)  AS input_tokens,
    SUM(output_tokens) AS output_tokens
  FROM api_calls
  WHERE session_id = ?
  GROUP BY bucket_ms
  ORDER BY bucket_ms ASC
`);
```

**Route handler:**
```js
fastify.get('/api/insights/tokens-over-time', (request, reply) => {
  const { session_id } = request.query;
  if (!session_id) return reply.code(400).send({ error: 'session_id required' });
  reply.send(stmtTokensOverTime.all(session_id));
});
```

Response shape: `[{ bucket_ms: 1741564800000, input_tokens: 1200, output_tokens: 340 }, ...]`

Note: `cache_read_tokens` and `cache_write_tokens` are excluded — charts show billed input/output only.
