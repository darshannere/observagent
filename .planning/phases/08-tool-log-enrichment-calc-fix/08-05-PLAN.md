Proximity matching strategy: Join api_calls to events by session_id and timestamp within 30 seconds (30,000 ms). Use a correlated subquery to find the nearest api_call:
  LEFT JOIN LATERAL (
    SELECT input_tokens, output_tokens FROM api_calls
    WHERE session_id = events.session_id
      AND ABS(timestamp_ms - events.timestamp) < 30000
    ORDER BY ABS(timestamp_ms - events.timestamp) ASC
    LIMIT 1
  ) ac ON 1=1

Note: SQLite does not support LATERAL joins. Use a correlated subquery instead:
  SELECT ...,
